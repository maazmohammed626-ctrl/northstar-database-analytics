# NorthStar Database and Analytics Assignment
This repository contains the datasets, notebooks, cleaned outputs, and MongoDB implementation for the NorthStar Urban Mobility and Logistics databases and analytics assignment.

## Folder Structure
- `northstar_dataset/` contains the original raw CSV datasets.
- `cleaned_dataset/` contains cleaned and processed CSV files exported from the Python notebook.
- `notebooks/` contains the SQL in R, R analytics, Python processing, and MongoDB PyMongo notebooks.

## Notebooks
1. `SQLR_21595984.ipynb`
2. `RANALYSISI_21595984.ipynb`
3. `PYTHON_DATA_21595984.ipynb`
4. `MONGODBPY_21595984.ipynb`

## Tools Used
- R
- SQL using sqldf
- Python Pandas and NumPy
- Matplotlib
- MongoDB Atlas
- PyMongo

## Purpose
The project analyses fragmented operational data from NorthStar and develops an integrated database and analytics solution to identify service inefficiencies, customer dissatisfaction patterns, delivery risks, and suitable NoSQL modelling approaches.
